# Marketing Kit

## One-Liner

Godot Launchpad is an open-source desktop version manager that installs,
configures, and launches Godot without sending you through a browser download
flow.

## Short Description

Godot Launchpad helps Godot developers switch versions without manual downloads,
zip extraction, or executable hunting. It checks stable Godot releases, installs
the selected version in the background, remembers your install folder and launch
arguments, and starts Godot from the managed install.

## Launch Post

I just open-sourced Godot Launchpad: a desktop version manager for Godot.

It checks stable Godot releases, installs your selected version in the
background, remembers your setup, and launches Godot directly. No browser
download flow, no manual extraction, no hunting for the executable.

It is early, macOS is the most tested path, and Windows/Linux testing would be
very welcome.

## Community Post

I built a small open-source desktop app for managing Godot versions:
Godot Launchpad.

The goal is to make Godot setup feel boring in a good way: choose a stable
release, let the app install/extract it in the background, and launch from a
managed install folder.

I would love feedback from Godot users, especially anyone who can test Windows
or Linux.
