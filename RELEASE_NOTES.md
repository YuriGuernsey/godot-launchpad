# Godot Launchpad v0.1.0

First public open-source release of Godot Launchpad.

## Highlights

- Checks the latest stable Godot releases from `godotengine/godot`.
- Installs the selected Godot version in the background.
- Extracts and configures a managed Godot executable automatically.
- Remembers install folder, launch arguments, and installed version paths.
- Launches Godot directly from the managed install.
- Includes initial macOS desktop packaging and project scaffolds for Windows and Linux.
- Adds the first Godot Launchpad logo and macOS app icon set.

## Status

This is an early release. macOS is the most tested path. Windows and Linux
testing, packaging help, and bug reports are welcome.
